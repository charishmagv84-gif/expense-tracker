# Expense Tracker (CLI)

A command-line Python application to track daily expenses.

## Features
- Add, view, and delete expenses
- Persistent storage using JSON (data survives program restarts)
- View total expenses
- Filter expenses by category
- Handles invalid input gracefully (error handling)

## How to run
python main.py

## What I learned
- File I/O and JSON serialization/deserialization
- Object-oriented programming (classes, __init__, __str__)
- Error handling with try/except
- List comprehensions
